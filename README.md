## Install dan Jalankan Aplikasi

Install aplikasi dan dependencies (cukup jalankan sekali):

```bash
1. go mod tidy
2. go mod download 
```

```bash
run mysql client di localhost terlebih dahulu
```

Jalankan aplikasi:

```bash
go run server.go
```
